# 웹사이트 개발 계획서

## 프로젝트 개요
- **프레임워크**: Next.js
- **배포**: Vercel
- **반응형**: 모바일 우선 풀 반응형
- **CI/CD**: GitHub Actions + Vercel 자동 배포

## 💡 핵심 추천사항

### Figma → 코드 변환 (가장 중요!)

- **3순위**: AI 도구 활용 (GPT-4V, Claude)

- 반응형은 모바일 우선 접근
